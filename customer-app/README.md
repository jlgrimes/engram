# Conch Customer App

Customer-facing web app for external users at `app.conch.so`.

## Run

```bash
cd customer-app
npm install
npm run dev
```

## Build

```bash
cd customer-app
npm run build
npm run start
```

## Scope

- Landing/offer hero
- Get Started CTA placeholder
- Pricing/offer placeholder section
- Trust/proof placeholder section
- Contact/onboarding placeholder form (no backend)
